## pioneer mini gui
gui for pioneer sdk

clone repo
```bash
git clone https://github.com/xrenvtomate/pioneer-gui
```

then 
```bash
cd client
npm i
```
to install dependencies
```bash
npm start
```
to start frontend

```bash
cd ../api
pip install flask uvicorn
```

```bash
uvicorn main:app --reload 
```
to start backend
